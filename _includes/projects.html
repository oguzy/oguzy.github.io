  VENGA (Very Native Galaxy Atmosphere) - VENGA is also my final project that i choosed at my university. I used google code to upload my codes. There is not a web site for the project yet. Shorty, it is a web site that reads the XML feed (blog entries) and decides the category of the entry by Bayesian Classify and shows it under some category. So a planet that enables the people read entries by cetagories. I used Python as a core language. Used feedparser to parse feeds and Zemberek to parse words according to the Turkish language rules. And i used Django to develop the web interface and database part.
